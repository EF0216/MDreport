## v68 weekly tabs by week_no 2026-05-10

- 重要、気温、売上、カテゴリ、ニュースに日別 / 週別切替を追加。
- 週別表示は `week_no` / `week` をもとに表示。
- 週タブは直近8週分。
- 今週が未完了の場合は、その日までの週累計で表示。
- 日別の選択肢は今週分を優先。
- 重要タブ内の売上動向アラートも週別切替に連動。
- カテゴリ表の週別時は「売上前週累計実績」「前週累計比」に見出しを切替。
